# Todo List Fullstack

Proyecto desarrollado para la materia de Programación Web.

## Nota sobre herramientas de apoyo

Se utilizó ChatGPT como herramienta de asistencia durante el desarrollo del proyecto, principalmente para la resolucion de dudas teoricas, depuracion de errores y mejora de fragmentos de código.

La logica e implementacion final fueron revisadas, integradas y validadas manualmente para asegurar el correcto funcionamiento del sistema.
